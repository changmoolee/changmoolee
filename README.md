# 👋 Hello, I'm Joseph!

## 🚀 About Me
안녕하세요! 저는 **프론트엔드 개발자**로 시작하여, **백엔드 및 데이터베이스**까지 경험하며 **풀스택 개발자로 성장 중**입니다.

💡 **배움과 성장**을 중요하게 생각하며, 실무에서 다양한 경험을 쌓으며 개발자로서의 깊이를 더해가고 있습니다.
